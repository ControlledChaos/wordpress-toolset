Tool HTML Buffer
===============================

Provides a filter for the whole HTML of a page.

## Filter 'toolset/tool_html_buffer/buffer'

````php
tool_add_filter( array(
	'tag' => 'toolset/tool_html_buffer/buffer',
	'callback' => function( $value ) {

		// filter something from $value

		return $value;
	}
) );
````
