Tool tool_nav_menus_register
===============================

This tool provides a function to register menus.

````php
	tool( array(
		'name' => 'tool_nav_menu_register',
		'param' => array(
			'location' => 'my-menu',
			'name' => 'My-Menu',
		)
	) );
````
