Tool
===============================

### HTML5

Adds HTML5 Theme support.
See: See: https://developer.wordpress.org/reference/functions/add_theme_support/

````php
	$GLOBALS['toolset'] = array(
		'inits' => array(
			'tool_html5' => true
		)
	);
````
