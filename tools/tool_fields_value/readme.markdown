Tool Fields Value
===============================

Provides a filter for the ACF field values.

## Filter 'toolset/tool_fields_value/value'

````php
tool_add_filter( array(
	'tag' => 'toolset/tool_fields_value/value',
	'callback' => function( $value ) {

		// filter something from $value

		return $value;
	}
) );
````
