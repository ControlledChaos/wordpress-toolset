Tool tool_meta_robots
===============================

This tool generates a `<meta name="robots" content="index, follow">` tag. Requires a ACF `page_meta_searchengine` multilang field.

````php
    tool( array(
        'name' => 'tool_meta_robots',
    ) );
````
